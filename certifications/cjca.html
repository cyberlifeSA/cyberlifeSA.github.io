Hack The Box CJCA

Junior Cybersecurity Analyst Certification

📌 Descripción

Hack The Box Certified Junior Cybersecurity Analyst (CJCA) es una certificación práctica orientada al desarrollo de habilidades fundamentales para un analista de ciberseguridad. Evalúa conocimientos mediante escenarios reales relacionados con análisis, reconocimiento, vulnerabilidades y seguridad ofensiva básica.

🎯 Conocimientos adquiridos

  • Reconocimiento y enumeración
  • Análisis de vulnerabilidades
  • Fundamentos de explotación
  • Seguridad web básica
  • Linux y Windows fundamentals
  • Análisis de servicios de red

🛡 Aplicación en SOC

Aunque está orientada desde una perspectiva técnica, permite comprender cómo los atacantes realizan reconocimiento y explotación. Este conocimiento ayuda a un analista SOC a identificar indicadores de compromiso y comprender técnicas utilizadas por actores maliciosos.

🔧 Herramientas utilizadas

Nmap Burp Suite Gobuster Metasploit Linux

📜 Evidencia

  • Certificación HTB CJCA
  • Labs completados
  • Writeups técnicos

💡 Reflexión personal

Esta certificación permitió comprender el ciclo de ataque desde una perspectiva práctica, fortaleciendo la capacidad de análisis y comprensión de técnicas utilizadas por atacantes.
