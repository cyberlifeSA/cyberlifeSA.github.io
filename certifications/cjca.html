Hack The Box Certified Junior Cybersecurity Analyst (CJCA)

Blue Team | Security Analysis | Junior Cybersecurity Analyst

📌 Descripción

Hack The Box Certified Junior Cybersecurity Analyst (CJCA) es una certificación práctica diseñada para validar conocimientos fundamentales en ciberseguridad mediante laboratorios y escenarios reales. Su enfoque combina conceptos de Blue Team y fundamentos ofensivos, permitiendo comprender cómo identificar vulnerabilidades, analizar sistemas y entender las técnicas empleadas por los atacantes para fortalecer la capacidad de detección y respuesta.

🎯 Conocimientos adquiridos

  • Reconocimiento y enumeración de sistemas.
  • Análisis de vulnerabilidades.
  • Fundamentos de seguridad en Linux y Windows.
  • Servicios y protocolos de red.
  • Conceptos básicos de Active Directory.
  • Seguridad Web.
  • Análisis de evidencias e indicadores de compromiso (IOC).
  • Metodologías de investigación.

🛡 Aplicación en un entorno SOC

Los conocimientos obtenidos permiten comprender cómo se desarrollan distintas fases de un ataque, facilitando la investigación de alertas y el análisis de eventos desde la perspectiva de un analista SOC. Comprender las técnicas utilizadas por un atacante ayuda a identificar comportamientos anómalos, reconocer indicadores de compromiso y realizar investigaciones más efectivas durante el proceso de respuesta a incidentes.

🔧 Herramientas utilizadas

Nmap Burp Suite Gobuster Metasploit Netcat Linux Windows HTTP DNS
