Splunk Cybersecurity Defense Analyst (CDSA)

SIEM | Threat Detection | Incident Investigation | Security Operations

📌 Descripción

Splunk Cybersecurity Defense Analyst (CDSA) es una certificación orientada al análisis de eventos de seguridad utilizando Splunk como plataforma SIEM. Su enfoque está en la investigación de incidentes, análisis de logs, detección de amenazas y aplicación de metodologías utilizadas por equipos Blue Team y Security Operations Center (SOC).

🎯 Conocimientos adquiridos

  • Arquitectura y funcionamiento de Splunk Enterprise.
  • Búsquedas y análisis mediante Splunk Processing Language (SPL).
  • Investigación de alertas e incidentes de seguridad.
  • Análisis de eventos provenientes de múltiples fuentes de datos.
  • Creación y optimización de consultas para detección de amenazas.
  • Correlación de eventos y análisis contextual.
  • Uso de dashboards e informes para monitoreo de seguridad.

🛡 Aplicación en un entorno SOC

Los conocimientos adquiridos permiten desempeñar tareas habituales de un analista SOC, incluyendo el monitoreo continuo de eventos, el análisis de alertas, la validación de posibles incidentes, la identificación de indicadores de compromiso (IOC) y el apoyo en procesos de respuesta ante incidentes. Esta certificación también fortalece la capacidad para trabajar con grandes volúmenes de registros (logs), priorizar eventos relevantes y documentar investigaciones siguiendo procedimientos operativos.

🔍 Competencias desarrolladas

  • Threat Detection
  • Threat Hunting
  • Incident Investigation
  • Log Analysis
  • IOC Analysis
  • Alert Triage

  • Security Monitoring
