Cisco CCST Cybersecurity

Cybersecurity Fundamentals | Entry-Level Security Certification

📌 Descripción

Cisco Certified Support Technician (CCST) Cybersecurity es una certificación enfocada en fundamentos esenciales de seguridad informática. Permite comprender conceptos relacionados con amenazas, vulnerabilidades, protección de sistemas, redes y principios utilizados dentro de equipos de seguridad.

🎯 Conocimientos adquiridos

  • Fundamentos de ciberseguridad
  • Tipos de amenazas y ataques
  • Conceptos de redes y protocolos
  • Seguridad de endpoints
  • Gestión básica de riesgos
  • Principios de defensa informática

🛡 Aplicación en un entorno SOC

Esta certificación proporciona una base sólida para comprender las alertas generadas por herramientas de seguridad. Ayuda a interpretar eventos relacionados con redes, usuarios, accesos y posibles comportamientos maliciosos, siendo una base importante para comenzar como SOC Analyst L1.

Networking Security Monitoring Firewalls Endpoint Security CIA Triad
